💸 Smart Financial Advisor
🚀 Your Personal Financial Data Explorer!

Get real-time market insights, stock quotes, news, and financial data with a powerful AI assistant. Powered by OpenAI's GPT-4o-mini and Polygon's API, this tool provides you with fast, reliable answers to all your financial queries.

🔥 Key Features:
Real-time Financial Tools: Access stock quotes, ticker news, financial reports, and aggregates.
Smart AI Agent: GPT-4 answers your questions by intelligently using financial tools.
Quick and Reliable: Instant responses with processing time displayed.

🛠️ How It Works:
Ask your financial question.

Agent Decision: The AI selects the best tool for your query.
Data Fetching: Polygon API provides accurate, up-to-date data.
Get Results: View the output and processing time.

💡 Get Started:
Clone the Repo and install dependencies:
pip install -r requirements.txt

Set up API Keys in .env file:
OPENAI_API_KEY=your_openai_key
POLYGON_API_KEY=your_polygon_key

Run the app:
python app.py
